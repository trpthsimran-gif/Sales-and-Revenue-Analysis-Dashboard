# Sales and Revenue Analysis Dashboard (Tableau)

This project is a **Tableau dashboard** created to analyze **sales quantity and revenue performance** across different countries.  
It helps identify **top-performing regions**, understand **revenue trends over time**, and visualize **geographic demand**.
